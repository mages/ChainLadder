# Extract residuals of a MackChainLadder model

Extract residuals of a
[`MackChainLadder`](http://mages.github.io/ChainLadder/reference/MackChainLadder.md)
model by origin-, calendar- and development period.

## Usage

``` r
# S3 method for class 'MackChainLadder'
residuals(object, ...)
```

## Arguments

- object:

  output of
  [`MackChainLadder`](http://mages.github.io/ChainLadder/reference/MackChainLadder.md)

- ...:

  not in use

## Value

The function returns a `data.frame` of residuals and standardised
residuals by origin-, calendar- and development period.
